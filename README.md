Project Description:
     A web-based Library Book Reservation System where students can browse books, reserve them online, and track their borrowings. Admin can manage books, users, and reservations.

Features:
 User(Index.HTML):
    User Registration & Login
    Browse Book Catalog
    Search books by title, author, genre, ISBN
    Filter books by genre
    Reserve a book online
    Cancel reservation
    User Dashboard with stats
    View active reservations
    View borrowed books
    View borrowing history
    View fines
 Admin(admin.html):
    Admin login protection
    Dashboard stats (Total Books, Users, Reservations, Fines)
    Add new books
    Delete books
    View all reservations
    Mark book as borrowed
    Mark book as returned
    Auto fine calculation for late returns
    View all registered users
